  <span>字符产生间距</span> 
  <span>字符产生间距</span> 
  <span>字符产生间距</span> 
  <span>字符产生间距</span> 
</div>

样式：

html{ 
  -webkit-text-size-adjust:none;/* 使用webkit的私有属性，让字体大小不受设备终端的调整，可定义字体大小小于12px */ 
} 
.span-wrap { 
  font-size:0;/* 所有浏览器 */ \*word-spacing:-1px;/\* 使用word-spacing 修复 IE6、7 中始终存在的 1px 空隙，减少单词间的空白（即字间隔） */ 
} 
.span-wrap span{ 
  font-size: 12px; letter-spacing: normal;/* 设置字母、字间距为0 */ 
  word-spacing: normal; /* 设置单词、字段间距为0 */ 
}
